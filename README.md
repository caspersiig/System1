# System1
Velkommen til vores 3 semsters sum projekt

Her er skallen til en smuk og eventyrlig hjemmeside og firestore database system lavet til en helt særlig madvogn.

---------------------------- VIGTIG INFORMATION TIL OPSÆTNING ----------------------------

Vi har i projektet indført en admin-side, der kan tilgås ved at skrive /admin i ens URL.
Her er der en login-knap, hvor man kan bruge Google Authentication til at 
logge ind og redigere i menuen, åbningstider og holdepladser.

Brugernavn og adgangskode er:

***HVIS MAN VIL FINDE LOGINOPLYSNINGER: SE .ENV FILEN***

Disse informationer er blevet gemt i en .env fil, som vi vil inkludere i vores afleverede mappe.
.Env-filen skal indsættes i VSCODE samme sted som vores menu-json eller .gitignore fil.

Herefter kan man logge ind med de givne informationer.
